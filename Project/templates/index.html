About Us

This is an NLP project created by 6 - AIDS students for assignment purposes under the guidance of Miss. Sandra Jeyes.

Implementation

Our Text Summarizer uses Natural Language Processing (NLP) techniques to generate concise summaries from input text. It includes features like:

  • Tokenization and Lemmatization using NLTK
  • TextRank algorithm for sentence extraction
  • Word cloud, bar chart, and pie chart visualizations
